Domanda 3: Si consideri il seguente codice

HTML / JS
<div align="center">
  <form name="modulo">
    <input type="text" name="over" value="Dove si trova il puntatore?" size="50px">
  </form>
  <a href="" onMouseOver="window.document.modulo.over.value='Destra';"
  onmouseout="window.document.modulo.over.value='Dove si trova il puntatore?'">scritta1</A>
  <BR><BR>
  <a href="#" onMouseOver="window.document.modulo.over.value='Sinistra';"
  onmouseout="window.document.modulo.over.value='Dove si trova il puntatore?'">scritta2</A>
</div>
  • 1. Intercetta il passaggio del mouse sopra ai link contenuti nella pagina. [cite: 405]
  • 2. Non mostra mai la scritta 'Sinistra'. [cite: 406]
  • 3. Mostra la scritta 'Destra' insieme alla scritta 'Sinistra' quando il mouse passa velocemente da un link all'altro. [cite: 407-408]
  • 4. Assegna unicamente uno di questi valori all'oggetto input: 'Destra' oppure 'Sinistra'. [cite: 409]

Domanda 4: Si consideri il seguente codice JavaScript

JAVASCRIPT
<script type="text/javascript">
  var listino = [{"id": 1, "prezzo": "500euro"}, { "id": 2, "prezzo": "400euro"}, {"id": 3, "prezzo": "600euro"}];
  var prodotti = [{"id": 1, "modello": "Galaxy S10", "marca": "Samsung"}, { "id": 2, "modello": "P20 pro", "marca": "Huawei"}, {"id": 3, "modello": "IphoneX", "marca": "Apple"}];
  
  sessionStorage.setItem("listino", JSON.stringify (listino));
  sessionStorage.setItem("prodotti", JSON.stringify (prodotti));
  
  var chiave = sessionStorage.key(1);
  var oggetto = JSON.parse (sessionStorage.getItem (chiave));
  alert(oggetto[1].modello);
</script>
  • 1. alert stampa 'P20 pro'. [cite: 417]
  • 2. sessionStorage.key(1) ritorna {'id': 2, "modello": "P20 pro", "marca": "Huawei"}. [cite: 418]
  • 3. JSON.parse trasforma una stringa in un oggetto JavaScript. [cite: 419]
