Domanda 3: Si consideri il seguente codice JavaScript [cite: 1052]

JAVASCRIPT
function exec_f(f, x) {
  return f(x);
}
alert(exec_f(new Function ("x", "return x*x"), 3));
  • 1. La funzione alert mostra undefined. [cite: 1057]
  • 2. La funzione alert mostra 9. [cite: 1058]
  • 3. La funzione alert mostra 3. [cite: 1059]

Motivare la risposta (max 3 righe): [cite: 1060] (La keyword "new Function" definisce dinamicamente una funzione. Passando x=3, ritornerà il quadrato 3*3 = 9)

Domanda 4: Si consideri il seguente codice HTML/JavaScript [cite: 1061]

HTML / JS
<p>Hello World!</p>
<p id="today">Today is AWC Examination Day!</p>
<p id="disclaimer"></p>
<p id="lunghezza"></p>

<script>
  var myNodeList = document.getElementsByTagName("p");
  var temp = myNodeList[1].innerHTML;
  
  document.getElementById("disclaimer").innerHTML = temp;
  document.getElementById("lunghezza").innerHTML = myNodeList.length;
  document.getElementsByTagName("p")[2].innerHTML = document.getElementById("lunghezza").innerHTML;
</script>
  • 1. Il paragrafo con id="disclaimer" e con id="lunghezza" hanno lo stesso valore. [cite: 1073]
  • 2. Il paragrafo con id="today" e con id="lunghezza" hanno lo stesso valore. [cite: 1074]
  • 3. Il paragrafo con id="disclaimer" ha valore 4. [cite: 1075]
  • 4. Il paragrafo con id="lunghezza" ha valore "Hello World!". [cite: 1076]
