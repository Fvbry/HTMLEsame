Domanda Aperta 1: Ruolo di Class e ID in CSS

Nell'ambito del linguaggio CSS si discutano in dettaglio il ruolo e le specifiche di class e id, spiegandone l'utilizzo e presentando un esempio concreto.

Soluzione:
In CSS, `class` e `id` sono attributi utilizzati per selezionare e stilizzare elementi HTML, ma con regole differenti.

- L'ID è un identificatore univoco: può essere assegnato a un solo elemento nella pagina (es. l'intestazione principale). Ha una priorità maggiore e nel CSS si richiama col prefisso `#`.
- La Class è riutilizzabile: può essere assegnata a molteplici elementi per condividere lo stesso stile (es. dei bottoni o delle card). Nel CSS si richiama col prefisso `.`.

Esempio concreto:
HTML: <div id="navbar"> <button class="btn">Home</button> </div>
CSS: #navbar { background: black; } .btn { color: red; }

Domanda Aperta 2: Scrivere un documento HTML

Scrivere un documento HTML... Lo schermo è diviso in tre zone. Nella parte in alto, 6 immagini piccole. Il click mostra la preview nella zona centrale. In basso un form di richiesta (Nome, Email, Indirizzo).

Soluzione:
HTML
<!DOCTYPE html>
<html>
<head>
  <title>Negozio</title>
  <style>
    .top { display: flex; justify-content: center; gap: 10px; }
    .mid { text-align: center; margin: 30px 0; }
    .bottom { text-align: center; }
  </style>
</head>
<body>
  <!-- Zona in Alto: Immagini -->
  <div class="top">
    <img src="thumb1.jpg" width="50" onclick="document.getElementById('preview').src='img1.jpg'">
    <img src="thumb2.jpg" width="50" onclick="document.getElementById('preview').src='img2.jpg'">
    <!-- (ripetuto per altre 4 immagini) -->
  </div>

  <!-- Zona Centrale: Preview -->
  <div class="mid">
    <img id="preview" src="default.jpg" width="300">
  </div>

  <!-- Zona in Basso: Form -->
  <div class="bottom">
    <form>
      Nome: <input type="text" name="nome"><br>
      Email: <input type="email" name="email"><br>
      Indirizzo: <textarea name="indirizzo"></textarea><br>
      <input type="submit" value="Invia Richiesta">
    </form>
  </div>
</body>
</html>
