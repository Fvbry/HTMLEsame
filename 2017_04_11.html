Domanda 1: Web Service vs Software-Oriented Architecture (SOA)

  • 1. Web service sono lo standard principale per realizzare SOA.
  • 2. SOA sono componenti loosely coupled.
  • 3. Web service è uno stile architetturale per implementare applicazioni software che usano servizi disponibili in rete.

Domanda 2: HTML DOM

  • 1. È uno standard che definisce la struttura di una pagina HTML.
  • 2. È usato da JavaScript per modificare gli elementi di una pagina HTML.
  • 3. Rappresenta una pagina HTML come un grafo ordinato aciclico.

Domanda 3: XML Remote Procedure Call (XML-RPC)

  • 1. Fornisce un protocollo di richiesta-risposta.
  • 2. Supporta tipi complessi come Array e Struct.
  • 3. È alla base dei Web Service REST.

JAVASCRIPT
<script>
  xmlhttp=new XMLHttpRequest();
  xmlhttp.open("GET", "menu.xml", false);
  xmlhttp.send();
  xmlDoc=xmlhttp.responseXML;
  
  xmlDocMenu=xmlDoc.getElementsByTagName("menu");
  
  document.write("<div>");
  document.write(xmlDocMenu[1].getElementsByTagName("primo")[2].childNodes[0].nodeValue+"<br>");
  document.write(xmlDocMenu[1].getElementsByTagName("secondo")[1].childNodes[0].nodeValue+"<br>");
  document.write(xmlDoc.getElementsByTagName("menu")[0].getElementsByTagName("dessert")[0].childNodes[0].nodeValue+"<br>");
  document.write("</div>");
</script>
  • 1. La funzione JavaScript stampa "risotto agli asparagi, parmigiana, frutta".
  • 2. La funzione JavaScript stampa "spaghetti allo scoglio, polpette, undefined".
  • 3. La funzione JavaScript stampa "risotto alla salsiccia, bistecca alla milanese, frutta".

Domanda 5: Si consideri il seguente codice CSS3

CSS
<style>
div {
  width: 100px;
  height: 100px;
  background-color: red;
  position: relative;
  animation-name: example;
  animation-duration: 4s;
  animation-iteration-count: infinite;
}
@keyframes example {
  0%   {background-color:red; left: 0px; top: 0px;}
  50%  {background-color:blue; left: 200px; top:200px;}
  100% {background-color: red; left: 0px; top: 0px;}
}
</style>
  • 1. Qualsiasi elemento div viene traslato avanti e indietro unicamente lungo l'asse delle x.
  • 2. Solo il primo elemento div definito all'interno della pagina HTML cambia colore.
  • 3. L'animazione viene eseguita una volta sola per 4 secondi.
  • 4. L'elemento div non cambia dimensione.
