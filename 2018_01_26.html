Domanda 1: Si consideri il seguente codice JavaScript

JAVASCRIPT
<script language="javascript">
  var f1 = "Ferrari";
  function funzionel (motogr) {
    return f1 + " " + motogr;
  }
  function funzione2() {
    var f1 = "Red Bull";
    return funzionel ("Yamaha");
  }
  window.alert(funzione2());
</script>
  • 1. funzione2 stampa "Ferrari Yamaha".
  • 2. funzione2 stampa "Red Bull Yamaha".
  • 3. funzione2 ritorna un errore.

Motivare la risposta (max 3 righe): (In JavaScript la funzione `funzionel` farà riferimento alla variabile globale `f1` valorizzata a "Ferrari" esistente al momento della sua definizione, ignorando la variabile locale `f1` "Red Bull" dichiarata dentro `funzione2`)

Domanda 3: Si consideri il seguente codice CSS

CSS
div {
  width: 100px;
  height: 100px;
  background: red;
  transition: width 2s, height 2s, transform 2s;
}
div:hover {
  width: 300px;
  height: 300px;
  transform: rotate (180deg);
}
  • 1. Definisce un'animazione basata su CSS3.
  • 2. Al passare del mouse sopra l'elemento div, sposta l'elemento div verso destra.
  • 3. Al passare del mouse sopra l'elemento div, ingrandisce l'elemento div. [cite: 923-924]
  • 4. Al passare del mouse sopra l'elemento div, ruota l'elemento div di 180 gradi. [cite: 924-925]
