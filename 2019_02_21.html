Domanda 5: Si consideri il seguente codice CSS3

CSS
<style>
div {
  position: relative;
  background-color: blue;
  width: 150px;
  height: 300px;
  animation-name: exam-trs;
  animation-duration: 10s;
}
@keyframes exam-trs {
  0%   {background-color: blue; left: 0px; top:0px;}
  25%  {background-color: red; left: 0px; top:200px; height: 150px;}
  50%  {background-color: green; left: 200px; top:200px;}
  75%  {background-color: red; left: 200px; top:0px;}
  100% {background-color: blue; left: 0px; top: 0px;}
}
</style>
  • 1. Qualsiasi elemento div viene traslato in senso antiorario. [cite: 477]
  • 2. Solo il primo elemento div definito all'interno della pagina HTML cambia colore. [cite: 478-479]
  • 3. L'animazione viene eseguita una volta sola per 10 secondi. [cite: 480]
  • 4. L'elemento div non cambia dimensione. [cite: 481]
