Domanda 2: Si consideri il seguente brano di codice XML

XML / SOAP
<soap:Envelope
xmlns:soap="http://www.w3.org/2003/05/soap-envelope/"
soap:encodingStyle="http://www.w3.org/2003/05/soap-encoding">
  <soap:Body>
    <m:GetPrice xmlns:m="https://www.awc.com/prices">
      <m:Item>One Plus 5</m:Item>
    </m:GetPrice>
  </soap:Body>
</soap:Envelope>
  • 1. È parte di una specifica di interfaccia WSDL.
  • 2. È parte di una richiesta SOAP.
  • 3. Viene usato da un web service che definisce interfacce standard HTTP.

Domanda 3: In un RESTful Web Service

  • 1. Le risorse vengono identificate attraverso file XML.
  • 2. Le risorse vengono accedute attraverso un'interfaccia uniforme basata su HTTP operation.
  • 3. Le operazioni GET, HEAD, PUT, DELETE sono idempotenti.
  • 4. Le operazioni PUT e POST hanno lo stesso comportamento.

Domanda 4: XML vs JSON

  • 1. La sintassi JSON è un sottoinsieme della sintassi JavaScript.
  • 2. XML è usato per descrivere dati strutturati che includono array.
  • 3. JSON è più verbose di XML.
  • 4. JSON è usato come base dei protocolli per i servizi web basati su SOAP.

Domanda 5: I fogli di stile CSS

  • 1. Definiscono lo stile di visualizzazione/formattazione.
  • 2. Utilizzano il selettore id per identificare lo stile di un singolo elemento all'interno della pagina HTML.
  • 3. Definiscono delle regole con la sintassi "selettore {proprietà: valore, ...}".

Domanda 6: HTML vs XHTML

  • 1. XHTML ha regole sintattiche più stringenti di HTML.
  • 2. XHTML e HTML sono molto diversi tra loro.
  • 3. XHTML combina HTML e JSON.
