Domanda 1: Si consideri il seguente codice JavaScript

JAVASCRIPT
<script language="javascript">
  var nome = "Claudio";
  function fz1 (cognome) {
    return nome + " " + cognome;
  }
  function fz2() {
    var nome = "Denise";
    return fz1("Ardagna");
  }
  window.alert(fz2());
</script>
  • 1. fz2 stampa "Claudio Ardagna". [cite: 1224]
  • 2. fz2 stampa "Denise Ardagna". [cite: 1225]
  • 3. fz2 ritorna un errore. [cite: 1226]

Domanda 2: Il selettore id di CSS

  • 1. Identifica lo stile di un singolo elemento all'interno della pagina HTML. [cite: 1228-1229]
  • 2. Permette di scegliere il nome del selettore da un vocabolario fisso di parole. [cite: 1230]
  • 3. È definito in un file CSS tramite la sintassi "#nomeselettore". [cite: 1231]
  • 4. Può essere combinato con selettori class. [cite: 1231]

Domanda 3: JSON notation

  • 1. Viene gestito in JavaScript tramite le funzioni JSON.stringify e JSON.parse. [cite: 1232-1233]
  • 2. Permette di formattare strutture dati. [cite: 1234]
  • 3. È più leggera di XML. [cite: 1235]
