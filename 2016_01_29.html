Domanda 5: Analisi Codice JS (Scope)

var globale = 5;
function fz1(locale) { return locale + globale; }
function fz2() { var globale = 30; return fz1(15); }
  • Risposta: 20 CORRETTA

Motivazione: JavaScript usa lo scope lessicale. La funzione fz1 fa riferimento alla variabile globale definita nel suo ambiente (5), ignorando la variabile locale di fz2.

Domanda Aperta: CSS Box Model

Soluzione: Il Box Model è il modello che definisce lo spazio occupato da un elemento. Si compone di Content (il contenuto), Padding (spazio interno tra contenuto e bordo), Border (il bordo stesso) e Margin (spazio esterno). La larghezza totale calcolata include la somma di tutte queste parti.
