Domanda 1: Si consideri il seguente codice HTML/JavaScript

HTML / JS
<div id="movies">
  <div style="margin:auto 0px">
    <h3 style="margin-left: 10px">MOVIES</h3>
    <div class="style" id="movie_1">
      <div class="image">
        <span class="center-img"></span>
        <img height="150px" src="static/img/doctorstrange.jpg">
      </div>
      <div>
        <br>
        <h3>Doctor Strange (2016)</h3>
        <ul>
          <li>1h 55m</li>
          <li>Action, Fantasy</li>
          <li>26 Ottobre 2016</li>
        </ul>
      </div>
    </div>
    <div><button onclick="hide('movie_1')">hide 1</button></div>
    <div class="movie" id="movie_2">[...]</div>
    <div><button onclick="hide('movie_2')">hide 2</button></div>
    <div class="movie" id="movie_3">[...]</div>
    <div><button onclick="hide('movie_3')">hide 3</button></div>
  </div>
</div>

<script type="text/javascript">
  function hide (div) {
    if (elem.style.visibility=="hidden")
      elem.style.visibility="visible";
    else
      elem.style.visibility="hidden";
  }
</script>

Supponendo che l'utente preme sul pulsante "Hide 2", quali delle seguenti immagini rappresenta la visualizzazione esatta?

  • 1. a
  • 2. b
  • 3. c.

Motivare la risposta (max 3 righe): (L'attributo visibility: hidden nasconde l'elemento ma lascia lo spazio che occupava originariamente nella pagina, a differenza di display: none)

Domanda 2: Quando diverse regole CSS si applicano a uno stesso elemento

  • 1. Viene applicata una valutazione a cascata per la risoluzione dei conflitti.
  • 2. Il primo ordinamento è fatto in base al peso e all'origine della specifica regola CSS.
  • 3. Priorità massima viene data alle regole definite dall'autore della pagina.
  • 4. Si considera sempre la regola più specifica.

Domanda 3: XML-RPC

  • 1. Forniscono un approccio per lo sviluppo di applicazioni distribuite.
  • 2. Forniscono tutte le funzionalità per una comunicazione sicura.
  • 3. Supportano i soli tipi complessi Array e Struct.
  • 4. Utilizzano il protocollo HTTP come mezzo di trasporto.

Domanda 5: Si consideri il seguente brano di codice JSON

JSON
{
  "team": [
    {
      "id": "001",
      "name": "Cleveland Cavaliers",
      "captain": "LeBron James",
      "city": "Cleveland"
    },
    {
      "id":"002",
      "name": "Golden State Warriors",
      "captain": "Stephen Curry",
      "city": "Oakland"
    }
  ]
}
  • 1. È in formato JSON.
  • 2. Specifica un array di team.
  • 3. Specifica un id di tipo intero.
  • 4. Specifica ogni singolo team come un array di quattro elementi di tipo stringa.
