Domanda 1: Si consideri il seguente codice HTML/JavaScript

HTML / JS
<div class="song">
  <button onclick="hide('song_1')">x</button>
  <div class="description" id="song_1">
    <!-- Contenuto omesso per brevità -->
  </div>
</div>
<script>
  function hide (target) {
    elem = document.getElementById(target);
    if (elem.style.display == "none")
      elem.style.display = "block";
    else
      elem.style.display = "none";
  }
</script>
  • 1. La funzione hide viene eseguita al caricamento della pagina HTML. [cite: 505]
  • 2. La funzione hide modifica una proprietà CSS. [cite: 506]
  • 3. elem.style.display="none" elimina l'elemento elem dalla pagina HTML lasciando tuttavia lo spazio occupato originariamente da elem. [cite: 507-508]
  • 4. document.getElementById(target) assegna alla variabile elem l'elemento con attributo id="target". [cite: 509]

Domanda 5: Si consideri il seguente codice CSS

CSS
#target {
  position: relative;
  padding: 10px;
  color: white;
  background-color: #b56457;
  height: 80px;
  width: 98%;
  display: inline-block;
}
  • 1. La proprietà padding definisce la distanza tra contenuto e bordo dell'elemento a cui viene applicato. [cite: 533]
  • 2. È un brano di codice HTML. [cite: 534]
  • 3. Definisce lo stile di visualizzazione per tutti gli elementi con class="target". [cite: 535-536]
  • 4. Può essere specificato nel tag <body> di un documento HTML. [cite: 537]
