Domanda 3: Si consideri il seguente codice CSS3 [cite: 303]

CSS
<style>
div {
  width: 100px;
  height: 100px;
  background-color: red;
  position: relative;
  animation-name: example;
  animation-duration: 4s;
  animation-iteration-count: infinite;
}
@keyframes example {
  0% {background-color:red; left: 0px; top:0px;}
  50% {background-color: blue; left: 200px; top:200px;}
  100% {background-color: red; left: 0px; top: 0px;}
}
</style>
  • 1. Qualsiasi elemento div viene traslato avanti e indietro sia sull'asse delle x che delle y. [cite: 324]
  • 2. Solo il primo elemento div definito all'interno della pagina HTML cambia colore. [cite: 325]
  • 3. L'animazione dura 4 secondi. [cite: 326]
  • 4. L'elemento div non cambia dimensione. [cite: 327]
