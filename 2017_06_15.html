Domanda 3: Applicazione REST (Ricette di cucina)

Si consideri un'applicazione REST di ricette di cucina che fornisce funzionalità per l'aggiunta di una ricetta con il seguente codice:

HTTP / JSON
HTTP_OPERATION URL
{
  "Titolo": "Pizza",
  "Ingredienti": [
    { "Nome": "Farina", "Quantita":"600g" },
    { "Nome": "Pomodoro", "Quantita": "300g" },
    { "Nome":"Mozzarella", "Quantita": "200g" }
  ],
  "Ricetta": [ ... ]
}

N.B. Dopo l'operazione di aggiunta ogni ricetta deve avere un id univoco.

Quale delle seguenti affermazioni è vera?

  • 1. HTTP_OPERATION=POST e URL=/ricetta/piattimici/id
  • 2. HTTP_OPERATION=PUT e URL=/ricetta/piattimici/
  • 3. HTTP_OPERATION=POST e URL=/ricetta/piattimici/

Motivare la risposta (max 3 righe): (L'operazione POST viene utilizzata quando il client chiede al server di creare una nuova risorsa subordinata all'URL specificato, lasciando al server il compito di assegnare un nuovo ID univoco).

Domanda 4: Si consideri il seguente codice CSS

CSS
<style>
p {
  margin: 100px 50px;
  padding: 25px 50px;
  border: 25px solid black;
}
</style>
  • 1. Lo stile viene applicato solo alla prima occorrenza di un elemento p.
  • 2. Personalizza il CSS box model, concetto che si applica a tutti gli elementi HTML.
  • 3. Definisce lo stile di due dei quattro lati del paragrafo.
  • 4. Padding rappresenta lo spazio libero attorno al contenuto nell'elemento p.
