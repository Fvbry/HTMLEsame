Domanda 2: Si consideri la seguente chiamata REST

HTTP / XML
PUT/blog/blog_id/pub/5
<titolo>A new assurance technique for the cloud</titolo>
<autore>Claudio Ardagna</autore>
<corpo>Description</corpo>
  • 1. Se pub con id 5 non esiste, lo crea. [cite: 354]
  • 2. Se pub con id 5 non esiste, l'azione eseguita è la stessa che verrebbe eseguita con un'operazione POST. [cite: 355-356]
  • 3. Se pub con id 5 esiste, non ha alcun effetto. [cite: 357]
  • 4. Se pub con id 5 esiste, il risultato è equivalente al risultato ottenuto con un'operazione POST. [cite: 358]

Domanda 4: Si consideri il seguente codice JavaScript

JAVASCRIPT
<script type="text/javascript">
  var stringa_globale = "Claudio";
  function fz1 (stringa_locale) {
    return stringa_locale + stringa_globale;
  }
  function fz2() {
    stringa_globale = "Erika";
    return fz1("Ciao ");
  }
  alert(fz2());
</script>
  • 1. fz2 stampa "Ciao Claudio". [cite: 374]
  • 2. fz2 stampa "Ciao Erika". [cite: 375]
  • 3. fz2 ritorna un errore. [cite: 376]
