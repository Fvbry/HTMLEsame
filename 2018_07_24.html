Domanda 1: I fogli di stile CSS

  • 1. Richiedono versioni HTML precedenti a HTML5.
  • 2. Definiscono lo stile di visualizzazione/formattazione.
  • 3. Definiscono delle regole con la sintassi "selettore [proprietà: valore, ...]".
  • 4. Utilizzano il selettore id per identificare lo stile di un singolo elemento all'interno della pagina HTML.

Domanda 2: Si consideri il seguente codice CSS

CSS
div {
  width: 200px;
  height: 200px;
  background: blue;
  transition: width 4s, height 2s, transform 2s;
}
div:hover {
  width: 400px;
  height: 400px;
  transform: rotate (90deg);
}
  • 1. Al passare del mouse sopra l'elemento div, ingrandisce l'elemento div.
  • 2. Definisce un'animazione basata su HTML5.
  • 3. Al passare del mouse sopra l'elemento div, inizia un'animazione della durata di 8 secondi.
  • 4. Al passare del mouse sopra l'elemento div, sposta l'elemento div verso destra.

Domanda 4: Si consideri il seguente codice JavaScript

HTML / JS
<FORM NAME="modulo" onSubmit="print(); return false;">
  <SELECT NAME="menu" id="menuid">
    <OPTION ID="op1" VALUE="stufato">Stufato di manzo</OPTION>
    <OPTION ID="op2" VALUE="tagliata">Tagliata</OPTION>
  </SELECT>
  <INPUT TYPE="Submit" VALUE="OK">
</FORM>

<SCRIPT language="JavaScript">
  var city = prompt("Sei carnivoro o vegetariano?", "Carnivoro");
  if(city == "Vegetariano") {
    window.document.modulo.menu.options[0].text="Minestra";
    window.document.modulo.menu.options[1].text="Patate al forno";
  }
  var select = document.getElementById('menuid');
  var opt = document.createElement('option');
  opt.id = "op3";
  opt.value = "pasta";
  opt.innerHTML = "Pasta aglio e olio";
  select.appendChild(opt);
</SCRIPT>
  • 1. Minestra in posizione 0 ove presente.
  • 2. Stufato di manzo e Tagliata sempre presenti tra le opzioni.
  • 3. Pasta aglio e olio sempre dopo Stufato di manzo.
  • 4. Pasta aglio e olio e Stufato di manzo mai insieme.
