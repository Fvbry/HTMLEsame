Domanda 1: Si consideri il seguente codice

JAVASCRIPT
<script type="text/javascript">
  var codicefiscale = [{"id": 1, "cf": "123456"},
                       {"id": 2, "cf": "234567"},
                       {"id": 3, "cf": "345678"}];
  var elenco = [{"id": 1, "nome": "Mario", "cognome": "Rossi"},
                {"id": 2, "nome": "Carlo", "cognome": "Bianchi"},
                {"id": 3, "nome": "Alice", "cognome": "Verdi"}];
  
  sessionStorage.setItem("cf",JSON.stringify(codicefiscale));
  sessionStorage.setItem("list", JSON.stringify(elenco));
  
  var chiave = sessionStorage.key(0);
  var tmp = sessionStorage.getItem(chiave);
  var tmp_parsed = JSON.parse(tmp);
  alert(tmp_parsed[1].cf);
</script>
  • 1. alert stampa 123456
  • 2. alert stampa 234567
  • 3. alert stampa 345678
  • 4. alert ritorna un errore

Motivare la risposta (max 3 righe): (In SessionStorage l'ordine delle chiavi non è garantito. Se la chiave(0) fosse "list", l'accesso a tmp_parsed[1].cf darebbe undefined poiché non esiste l'attributo "cf" nell'array "elenco"). [cite: 1447-1471]

Domanda 2: Il seguente brano di codice

CSS
div {
  width: 100px;
  height: 100px;
  background: red;
  transition: width 2s, height 2s, transform 2s;
}
div:hover {
  width: 300px;
  height: 300px;
  transform: rotate (180deg);
}
  • 1. Definisce un'animazione basata su CSS3
  • 2. Al passare del mouse sopra l'elemento div, sposta l'elemento div verso l'alto
  • 3. Al passare del mouse sopra l'elemento div, rimpicciolisce l'elemento div
  • 4. Al passare del mouse sopra l'elemento div, ruota l'elemento div di 180 gradi [cite: 1478-1494]
