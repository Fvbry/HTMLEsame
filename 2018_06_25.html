Domanda 3: In un RESTful Web Service

  • 1. Vengono definite funzioni tramite interfacce uniformi.
  • 2. Le risorse vengono identificate tramite URI.
  • 3. Ogni servizio può usare sia rappresentazioni JSON che XML.

Domanda 4: XML vs JSON

  • 1. JSON è usato per descrivere dati strutturati che includono array.
  • 2. JSON è più verboso di XML.
  • 3. XML è usato come base dei protocolli per i servizi web basati su SOAP.

Domanda 5: CSS box model

  • 1. Racchiude tutti gli elementi HTML in un box.
  • 2. Specifica unicamente margin, border, padding.
  • 3. Permette di definire stile, spessore e colore del margin.
  • 4. Impone lo stesso formato per tutti e quattro i lati della box.
