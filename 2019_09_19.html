Domanda 1: Si consideri il seguente brano di codice

HTML / JS
<!DOCTYPE html>
<html>
<body>
<h1>September Exam</h1>
<div>
  <b>Student: </b> <span id="std"></span><br>
  <b>Day: </b> <span id="d"></span><br>
  <b>Hour:</b> <span id="h"></span>
  <b>Project: </b> <span id="p"></span>
</div>
<script>
  var xmlhttp, xmlDoc;
  xmlhttp = new XMLHttpRequest();
  xmlhttp.open("GET", "URL", false);
  xmlhttp.send();
  xmlDoc = xmlhttp.responseXML;
  document.getElementById("std").innerHTML= xmlDoc.getElementsByTagName("std") [0].childNodes[0].nodeValue;
  document.getElementById("d").innerHTML= xmlDoc.getElementsByTagName("d") [0].childNodes[0].nodeValue;
  document.getElementById("h").innerHTML= xmlDoc.getElementsByTagName("h") [0].childNodes[0].nodeValue;
  document.getElementById("p").innerHTML= xmlDoc.getElementsByTagName("p") [0].childNodes[0].nodeValue;
</script>
</body>
</html>
  • 1. Dopo l'esecuzione dello script, i tag span sono riempiti con valori recuperati in locale.
  • 2. Il terzo parametro della funzione xmlhttp.open indica una richiesta sincrona.
  • 3. xmlhttp.responseXML contiene il risultato della richiesta effettuata.
  • 4. XMLHttpRequest permette di scambiare anche documenti JSON.

Domanda 3: Un selettore di una regola CSS

  • 1. Può essere qualunque tag HTML.
  • 2. Può riferirsi a una classe di tag.
  • 3. Può avere un nome diverso da quello di un tag HTML.

Domanda 4: JSON

  • 1. È un dialetto XML.
  • 2. È uno standard progettato per lo scambio di dati.
  • 3. È leggibile sia dalla macchina che dall'uomo.
  • 4. È indipendente dal linguaggio.
